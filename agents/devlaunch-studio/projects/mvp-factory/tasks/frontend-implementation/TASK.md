## Scope

Implement all React/Next.js UI components from PixelCraft's Figma designs, wired to BackendAce's APIs.

## Deliverables

1. All screens and user flows implemented from Figma specifications
2. API integration: all backend endpoints consumed via typed client
3. State management for user sessions and data flows
4. Responsive layouts: mobile (375px), tablet (768px), desktop (1280px)
5. Frontend unit and component tests with ≥ 80% coverage
6. Accessibility audit: WCAG 2.1 AA compliance
7. Core Web Vitals targets met: LCP < 2.5s, FID < 100ms, CLS < 0.1

## Success Criteria

- All Figma designs implemented pixel-accurately (spot check by PixelCraft)
- All API integrations working against staging backend
- Frontend test coverage ≥ 80%
- No accessibility violations at WCAG 2.1 AA level
- Deployed to staging and approved by LaunchCEO for visual review
