Conduct the weekly sprint review and prepare the client-facing delivery report.

## Checklist

1. Review all tasks completed this week against sprint goals
2. Calculate sprint velocity: story points completed vs planned
3. Assess sprint health: on track / at risk / off track for 14-day delivery
4. Identify scope creep or requirement changes introduced this week
5. Prepare demo notes for any completed features ready for client review
6. Draft weekly status report for LaunchCEO: progress, risks, recommendations
7. Post report as task comment and @-mention LaunchCEO for review
