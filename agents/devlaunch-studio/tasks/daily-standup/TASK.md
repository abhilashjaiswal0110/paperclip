Run the daily team standup and update the sprint tracking board.

## Checklist

1. Review all in-progress Paperclip tasks across the team
2. For each active task: confirm status, identify blockers, update estimate to completion
3. Flag any tasks at risk of missing their sprint deadline
4. Identify any cross-team dependencies that need coordination (e.g. BackendAce API not ready for FrontForge)
5. Post standup summary as a comment: what's done, in-progress, blocked, at-risk
6. Escalate any sprint-threatening blockers to ArchLead immediately
