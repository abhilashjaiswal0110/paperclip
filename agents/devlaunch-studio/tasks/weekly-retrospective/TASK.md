Run the weekly retrospective for DevLaunch Studio and deliver action items for the following week.

## Agenda

1. **What went well?** — Pull completed tasks from the past 7 days. Identify what worked.
2. **What didn't go well?** — Pull blocked, failed, or escalated tasks. Root cause each one.
3. **What should we improve?** — Identify one process, workflow, or tool improvement for the next sprint.
4. **Action items** — Create specific, assigned Paperclip tasks for each improvement identified.
5. **Budget check** — Review weekly spend vs budget allocation for each active agent.

## Output

A retrospective summary posted as a task comment, with:
- Completed tasks count (Green)
- Blocked or escalated tasks with root causes (Red/Yellow)
- One improvement action item created for the following week
- Budget spend summary for the week
