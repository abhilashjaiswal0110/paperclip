## Scope

Build and maintain the living API catalogue — the authoritative index of all APIs delivered by APIConnect Services, with version status, SLA metrics, and deprecation timelines.

## Deliverables

1. API catalogue structure: API name, version, status (active/deprecated/sunset), SLA metrics, owner
2. Version roadmap integration: planned future versions with target dates
3. Deprecation tracker: deprecated endpoints with sunset date and migration guide links
4. Client-API mapping: which clients consume which APIs (for breaking change impact analysis)
5. Delivery metrics integration: turnaround time, SLA adherence per API

## Success Criteria

- Catalogue covers 100% of delivered APIs
- Version roadmap updated at least monthly
- Deprecation notices trigger automatically 90 days before sunset dates
- ConnectCEO can generate a portfolio report from the catalogue at any time
