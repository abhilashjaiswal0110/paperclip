Run the daily delivery standup and update engagement tracking.

## Checklist

1. Review all active integration engagements against 5-day turnaround SLA
2. Flag any engagement at day 3 or beyond that is not yet in testing
3. Identify cross-team blockers (e.g. spec not yet approved, gateway config pending)
4. Update engagement status in the delivery tracker
5. Post standup summary as task comment: in-flight engagements, at-risk items, blockers
6. Escalate any engagement at risk of missing 5-day SLA to APIArchCTO immediately
