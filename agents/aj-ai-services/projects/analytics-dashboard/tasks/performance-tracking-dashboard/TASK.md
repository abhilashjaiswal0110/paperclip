## Objective

Build the weekly performance tracking report that AnalyticsEngine delivers every Monday.

## Acceptance Criteria

- [ ] Covers all posts published in the prior 7 days
- [ ] Metrics per post: impressions, engagement rate, reach, follower delta, click-through rate
- [ ] Trend comparison: this week vs prior 4-week moving average
- [ ] Top 3 and bottom 3 posts with root-cause analysis (format, topic, posting time, hashtag count)
- [ ] Audience growth rate: new followers this week vs prior 4-week average
- [ ] Report posted as Paperclip task comment and @-mentions SocialSage and AJ
