## Scope

Integrate with LinkedIn Analytics API and build the automated post-performance tracking pipeline.

## Deliverables

1. LinkedIn Analytics API integration with automated data pull after each post
2. Performance metrics calculation: impressions, engagement rate, click-through, follower growth
3. Post-performance tracking at 24h, 72h, and 7 days after publication
4. Performance dashboard with trend charts
5. Weekly analytics digest template and automated delivery

## Success Criteria

- Analytics data pulls automatically for every post published by PublishBot
- Engagement rate calculated consistently using (likes + comments + shares) / impressions
- Weekly digest delivered every Monday to SocialSage and AJ
