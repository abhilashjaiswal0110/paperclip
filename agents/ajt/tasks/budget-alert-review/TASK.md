Review weekly AI API spend and infrastructure costs to ensure the company stays within budget thresholds.

## Checklist

1. Pull the weekly cost report from the Anthropic console for all agents
2. Pull infrastructure cost summary (cloud, CI/CD, hosting)
3. Compare total spend against monthly budget allocation
4. Flag any agent that exceeded its weekly token budget by >20%
5. If total burn rate exceeds monthly runway, escalate to CTO for optimisation
6. Review whether cheap model profile agents can absorb more workload to reduce costs
7. Log spend summary and any decisions to CEO memory files
8. If runway falls below 8 weeks, convene emergency budget review with all leads
