## Objective

Establish full cloud cost visibility across all managed client environments, implement tagging governance, create the cost baseline, and produce the first rightsizing and Reserved Instance recommendations that launch the 30% cost reduction programme.

## Acceptance Criteria

- [ ] Cloud cost exports enabled for all accounts/subscriptions/projects (AWS Cost & Usage Report, Azure Cost Management, GCP Billing Export → BigQuery)
- [ ] Cost allocation tags enforced: `Environment`, `Owner`, `CostCentre` on 100% of resources
- [ ] Tagging compliance report automated: weekly scan, resources missing tags escalated to ArchBot
- [ ] Per-client cost baseline established: 3-month trailing average by service, environment, and team
- [ ] Budget alerts configured: 80% soft alert (notify CostSage), 100% hard alert (notify CloudCEO)
- [ ] Cost anomaly detection enabled: daily spend deviation > 20% triggers alert to CostSage within 4 hours
- [ ] Rightsizing analysis completed: all instances analysed for CPU/memory utilisation over 30 days; recommendations produced with estimated monthly savings
- [ ] Reserved Instance / Savings Plan coverage analysis completed: identify top-20 on-demand spend candidates
- [ ] First rightsizing recommendations documented and assignee set to ArchBot for implementation
- [ ] 30% cost reduction roadmap drafted: quarterly milestones, levers, and owners
- [ ] Monthly cost report template created and validated (will be auto-generated by weekly-cost-report routine)
- [ ] Storage tier optimisation analysis: S3/blob/GCS objects > 90 days old in hot tier identified for lifecycle policy

## Cost Reduction Roadmap (initial)

| Quarter | Target Reduction | Primary Lever |
|---------|-----------------|--------------|
| Q1 | 10% | Rightsize oversized instances + eliminate idle resources |
| Q2 | 20% | Reserved Instance / Savings Plan purchases |
| Q3 | 28% | Spot/preemptible for batch workloads + storage tiering |
| Q4 | 30%+ | Architectural optimisation (data transfer, caching) |

## Security Requirements

- Cloud billing API credentials scoped to read-only billing access
- Cost reports containing client billing data marked confidential
- Budget alert thresholds and client cost data retained for 13 months for audit
